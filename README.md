# Rock Paper Scissors Game ✊✋✌️

A simple console-based Rock Paper Scissors game built in Python where the user plays against the computer.

## How to Play
- The user chooses Rock, Paper, or Scissors
- The computer randomly selects its choice
- The program compares both choices and declares the winner

## Technologies Used
- Python

## How to Run
```bash
python rock_paper_scissors.py
